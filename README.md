# BabyCareApp

BabyCareApp is a release-ready Flutter application for mothers and child care. The app provides a mobile interface for parenting support, baby care guidance, health-related services, shopping categories, appointments, activity tracking, and user authentication.

## Features

- Splash screen and onboarding flow
- Login and registration screens
- Firebase Authentication integration
- Dashboard with main app sections
- Child care advice and article details
- Health, hygiene, nutrition, clothes, toys, and accessories categories
- Appointment and doctor detail screens
- Shopping cart flow
- Activity log screen
- Profile screen
- Local image assets and JSON data
- Material 3 based UI theme
- Android, iOS, Web, macOS, Linux, and Windows Flutter project structure
